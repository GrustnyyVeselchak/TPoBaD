# Repos searcher
### Предназначен для поиска и вывода списка репозиториев по имени пользователя
Отображаются следующие данные репозитория:
- Название
- Описание
- Колличество звезд
- кнопка по нажатию на которой откроется сама репозитория

## Запуск
После копирования репозитории используем команду для установки зависимостей

```pnpm
pnpm install
```

далее запускаем проект

```pnpm
pnpm run dev
```

в браузере заходим по URL: 
  http://localhost:5173/
