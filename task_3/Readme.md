# Задача: реализовать контролируемый React-компонент AsyncSearch, который:
- принимает onSearch(query: string): Promise<string[]> — функцию для получения результатов;
- имеет поле ввода; ввод дебаунсится на 300ms;
- при каждом новом запросе предыдущий должен быть отменён (если возможно);
- показывает "Loading..." при ожидании;
- отображает список результатов и обработку ошибок;
- Ограничения: реализовать без внешних библиотек; можно использовать fetch/AbortController или промисы.

## Критерии приёмки:
- дебаунс работает (300ms);
- при быстром вводе предыдущие запросы не влияют на UI (отменяются или игнорируются);
- корректно обрабатываются ошибки;
- код читабелен и типизирован.

## Стек:
- bun
- Vite

- Type Script
